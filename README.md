# ☀️ Времето в България — Weather App (Course Project)

Малко уеб приложение за проверка на **текущото време** в населени места в България.  
Използва **Nominatim (OpenStreetMap)** за геокодиране и **Open-Meteo** за метео данни.

---

## ✨ Функции

- 🔎 Търсене по населено място (на български)  
- 🗺️ Намира координати (lat/lon) чрез Nominatim
- 🌡️ Показва текуща температура + 💨 вятър и посока
- 📱 Mobile-friendly дизайн:
  - странични менюта като “drawers” (с плъзгане) за мобилни/таблет
  - backrop overlay + блокиране на scroll при отворено меню
- 🎥 Втора страница: **AI Видео Галерия** (учебна демонстрация)

---

## 🧱 Технологии

- HTML5 / CSS3 (Responsive)
- JavaScript (Vanilla)
- APIs:
  - Nominatim (OpenStreetMap) — геокодиране
  - Open-Meteo — текущо време
 
- AI:
ChatGPT 5 - сценарий на прогноза
